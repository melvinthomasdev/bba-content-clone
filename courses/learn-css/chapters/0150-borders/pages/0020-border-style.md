The `border-style` property is used
to identify the type of border:

<codeblock language="css" type="lesson">
<code>
<panel language="html">
<div id="first">
  Solid border
</div>
<div id="second">
  Dashed border
</div>
<div id="third">
  Double border
</div>
</panel>
<panel language="css">
div {
  padding: 10px;
  margin: 10px;
}
#first {
  border-style: solid;
}
#second {
  border-style: dashed;
}
#third {
  border-style: double;
}
</panel>
</code>
</codeblock>

Values for `border-style` include `solid`, `dashed`, `double`, `dotted`, `groove`, `ridge`, `inset`, `outset` and `none`. Try them out in the code editor given above.
