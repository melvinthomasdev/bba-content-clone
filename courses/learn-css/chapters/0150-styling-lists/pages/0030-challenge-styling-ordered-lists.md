Change the marker of the list items to `upper-roman`:

<codeblock language="css" type="exercise" testMode="fixedInput">
<code>
<panel language="html">
<h3>
  Planets of the Solar system
</h3>
<ol>
  <li>Mercury</li>
  <li>Venus</li>
  <li>Earth</li>
  <li>Mars</li>
  <li>Jupiter</li>
  <li>Saturn</li>
  <li>Uranus</li>
  <li>Neptune</li>
</ol>
</panel>
<panel language="css">

</panel>
</code>

<solution>
ol {
  list-style-type: upper-roman;
}
</solution>
</codeblock>
