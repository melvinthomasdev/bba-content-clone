The `margin` properties
can also accept
negative values.
This reduces the spacing around elements.

<codeblock language="css" type="lesson">
<code>
<panel language="html">
<div id="first">
</div>
<div id="second">
</div>
</panel>
<panel language="css">
#first {
  background-color: orange;
  height: 100px;
}
#second {
  background-color: green;
  width: 50%;
  height: 100px;
  margin-top: -2em;
}
</panel>
</code>
</codeblock>
