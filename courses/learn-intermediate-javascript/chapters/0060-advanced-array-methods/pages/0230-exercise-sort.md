Use the `sort()` method
to sort the `names` array
in alphabetical order.

<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const names = ["Eve", "Sam", "Oliver", "Adam"];

console.log(names);
</code>

<solution>
const names = ["Eve", "Sam", "Oliver", "Adam"];
names.sort();
console.log(names);
</solution>
</codeblock>
