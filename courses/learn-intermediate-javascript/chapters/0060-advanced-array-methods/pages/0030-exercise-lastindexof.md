Use the `lastIndexOf()` method
to find the last index of **"Sam"**
in the `names` array.

<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const names = ["Adam", "Sam", "Eve", "Oliver", "Sam"];
console.log();
</code>

<solution>
const names = ["Adam", "Sam", "Eve", "Oliver", "Sam"];
console.log(names.lastIndexOf("Sam"));
</solution>
