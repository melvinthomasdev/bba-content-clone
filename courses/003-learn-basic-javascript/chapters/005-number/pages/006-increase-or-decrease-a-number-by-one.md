We can use `++` to increment
the value of a number by `1`.

<Editor lang="javascript">
<code>
let number = 10;
number++; // This is same as number = number + 1

console.log(number);
</code>
</Editor>

We can use `--` to decrement
the value of a number by `1`.

<Editor lang="javascript">
<code>
let number = 10;
number--; // This is same as number = number - 1

console.log(number);
</code>
</Editor>
