The code given below displays
`Welcome` in the console.
Enclose the `console.log` statement
in a **for loop**,
such that it displays `Welcome` **3** times.

<Editor lang="javascript" type="exercise">
<code>
console.log("Welcome");
</code>

<solution>
for (let i = 0; i < 3; i++) {
  console.log("Welcome");
}
</solution>
</Editor>
