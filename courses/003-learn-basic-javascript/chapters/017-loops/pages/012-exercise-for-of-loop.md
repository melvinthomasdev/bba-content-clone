Use a `for...of` loop to print all the **values** of the `fruits` array.

Use `console.log` to print the output.

<Editor lang="javascript" type="exercise">
<code>
let fruits = [
  "Apple",
  "Orange",
  "Banana"
]
</code>

<solution>
let fruits = [
  "Apple",
  "Orange",
  "Banana"
]

for (let fruit of fruits) {
  console.log(fruit);
}
</solution>
</Editor>
