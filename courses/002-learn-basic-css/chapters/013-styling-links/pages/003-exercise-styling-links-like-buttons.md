Make the link given below
look like a button by adding
padding of `10px`,
background color of `cyan`
and
border radius of `5px`:

<Editor lang="css" type="exercise">
<code>
<panel lang="html">
<h2>Button Link</h2>
<a href="#">
  Click me
</a>
</panel>
<panel lang="css">
a {
  font-family: sans-serif;
  color: darkcyan;
  text-decoration: none;
}
</panel>
</code>

<solution>

  font-family: sans-serif;
  color: darkcyan;
  text-decoration: none;
  padding: 10px;
  background-color: cyan;
  border-radius: 5px;
}
</solution>
