Shift the image from the left by
**50%** using `position:relative`
and the `left` property:

<Editor lang="css" type="exercise">
<code>
<panel lang="html">
<img src="munnar-kerala-06.jpg" alt="Munnar Kerala">
</panel>
<panel lang="css">
img {
  width: 50%;
}
</panel>
</code>

<solution>
img {
  width: 50%;
  position: relative;
  left: 50%;
}
</solution>
</Editor>
