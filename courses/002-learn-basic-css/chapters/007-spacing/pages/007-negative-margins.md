The `margin` properties
can also accept
negative values.
This reduces the spacing around elements.

<Editor lang="css">
<code>
<panel lang="html">
<div id="first">
</div>
<div id="second">
</div>
</panel>
<panel lang="css">
#first {
  background-color: orange;
  height: 100px;
}
#second {
  background-color: green;
  width: 50%;
  height: 100px;
  margin-top: -2em;
}
</panel>
</code>
</Editor>
