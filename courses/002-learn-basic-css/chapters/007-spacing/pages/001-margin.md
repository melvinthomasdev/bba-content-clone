The `margin` property
defines the spacing
**around** an element.

<Editor lang="css">
<code>
<panel lang="html">
<p>
  A paragraph with 2em margin on all four sides
</p>
<span>
  Another element here
</span>
</panel>
<panel lang="css">
body {
  background-color: orange;
}
p {
  background-color: yellow;
  margin: 2em;
}
</panel>
</code>
</Editor>

Try changing the `margin` to `0em` or `5em` to notice the difference.
