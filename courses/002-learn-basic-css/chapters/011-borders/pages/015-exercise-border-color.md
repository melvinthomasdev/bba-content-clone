Set a `groove` border with `#008B8B` colour on the `p` element:

<Editor lang="css" type="exercise">
<code>
<panel lang="html">
<p>
  "You can get everything you want in life if you will just help other people get what they want."
  <br>
  -- Zig Ziglar
</p>
</panel>
<panel lang="css">
p {
  padding: 15px;
}
</panel>
</code>

<solution>
p {
  padding: 15px;
  border-style: groove;
  border-color: #008B8B;
}
</solution>
</Editor>
