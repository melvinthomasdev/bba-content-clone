**`font-size`** property is used to set the size of
the font. The default font size set in browsers is **16px**.

<codeblock language="css" type="lesson">
<code>
<panel language="html">
<p>The default font size is 16px</p>
</panel>
<panel language="css">
p {
  font-size: 100%;
}
</panel>
</code>
</codeblock>
