We can add nearly all
HTML tags within
`<li>` elements.

This can be done in both
ordered
and
unordered lists:

<codeblock language="html" type="lesson">
<code>
<h1>Menu</h1>
<ul>
  <li><h3>Home</h3></li>
  <li><h3>About</h3></li>
  <li><h3>Courses</h3></li>
</ul>

<h2>Ice cream Flavors</h2>
<ol>
  <li><p>Vanilla</p></li>
  <li><p>Chocolate</p></li>
  <li><p>Strawberry</p></li>
</ol>
</code>
</codeblock>
