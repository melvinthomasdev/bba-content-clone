The `<caption>` element is used to add a title to the table. It must be placed immediately after the opening `<table>` tag.

<Editor lang="html">
<code>
<table>
  <caption>Students</caption>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Gender</th>
  </tr>
  <tr>
    <td>Oliver</td>
    <td>25</td>
    <td>Male</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>30</td>
    <td>Female</td>
  </tr>
</table>
</code>
</Editor>
