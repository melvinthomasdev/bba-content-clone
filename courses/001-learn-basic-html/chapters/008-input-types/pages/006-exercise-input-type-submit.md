Create an input of type **submit** in the form, below the other input element:

<Editor lang="html" type="exercise">
<code>
<form>
  <label>Password: </label>
  <input type="password">
</form>
</code>

<solution>
<form>
  <label>Password: </label>
  <input type="password">
  <input type="submit">
</form>
</solution>
</Editor>
