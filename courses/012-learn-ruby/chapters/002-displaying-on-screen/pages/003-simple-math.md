Ruby can do simple math.

You can see the result using `puts`.

Run the code to see the result for addition:

<Editor lang="ruby">
<code>
puts 10 + 20
</code>
</Editor>

Subtraction:

<Editor lang="ruby">
<code>
puts 30 - 10
</code>
</Editor>

Multiplication:

<Editor lang="ruby">
<code>
puts 8 * 7
</code>
</Editor>

Division:

<Editor lang="ruby">
<code>
puts 30/6
</code>
</Editor>
