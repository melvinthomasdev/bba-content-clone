In this exercise, add to the
hash, a key of **height** and
a value **6 ft**.

<Editor lang="ruby" type="exercise">
<code>
hash = {
  "first_name" => "John",
  "last_name" => "Smith",
  "age" => 25
}
</code>

<solution>
hash = {
  "first_name" => "John",
  "last_name" => "Smith",
  "age" => 25
}

hash["height"] = "6 ft"

puts hash
</solution>
</Editor>
