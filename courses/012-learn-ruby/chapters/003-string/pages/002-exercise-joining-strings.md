The code editor uses
two `puts` statements to
display a message.

Instead of using two
`puts` statements,
concatenate the two strings
and
then print the result
using a single `puts` statement.

<Editor lang="ruby" type="exercise">
<code>
puts "I love"
puts "Spiderman"
</code>

<solution>
puts "I love " + "Spiderman"
</solution>
</Editor>
