A single `=` is used to assign
value to a variable.

```
name = "John Smith"
```

A double `==` is used to check equality.
It checks whether or not
the value on
the left side is same
as the value on the right
side.

It is used in a lot in 
`if else` conditions.

<Editor lang="ruby">
<code>
name = "John Smith"
if name == "John Smith"
  puts "I'm John"
end
</code>
</Editor>
