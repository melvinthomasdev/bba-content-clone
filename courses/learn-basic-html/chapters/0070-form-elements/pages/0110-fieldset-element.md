The `<fieldset>` element groups related form elements. Inside the `<fieldset>`, a `<legend>` element sets the caption at the top.

<codeblock language="html" type="lesson">
<code>
<fieldset>
  <legend>Location:</legend>
  <label>City:</label>
  <input type="text">
  <br>
  <label>State: </label>
  <input type="text">
  <br>
  <label>Country: </label>
  <input type="text">
</fieldset>
</code>
</codeblock>
