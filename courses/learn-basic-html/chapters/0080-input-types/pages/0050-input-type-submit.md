`<input type="submit">` creates a **button** for submitting the form data.

<codeblock language="html" type="lesson">
<code>
<form>
  <label>Username:</label>
  <input type="text">
  <br>
  <label>Password:</label>
  <input type="password">
  <br>
  <input type="submit">
</form>
</code>
</codeblock>
