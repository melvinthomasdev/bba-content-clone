Use the `indexOf()` method
to search for **Oliver**
in the **names** array.
The search should
start from index **1**.

<Editor lang="javascript" type="exercise">
<code>
let names = ["Adam", "Eve", "Oliver", "Sam", "Oliver"];
console.log();
</code>

<solution>
let names = ["Adam", "Eve", "Oliver", "Sam", "Oliver"];
console.log(names.indexOf("Oliver", 1));
</solution>
