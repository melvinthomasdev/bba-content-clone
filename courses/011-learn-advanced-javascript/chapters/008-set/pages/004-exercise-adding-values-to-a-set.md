Use `set.add()` to add
`apple`, `banana`
and
`orange` to the `fruits` Set.

<Editor type="exercise" lang="javascript">
<code>
let fruits = new Set();

console.log(fruits);
</code>

<solution>
let fruits = new Set();

fruits.add("apple");
fruits.add("banana");
fruits.add("orange");

console.log(fruits);
</solution>
</Editor>
