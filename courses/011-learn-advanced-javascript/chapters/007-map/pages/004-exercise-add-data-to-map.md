Use the `map.set()` method to
store the key `fruitName`
and the value `apple` in the
`fruits` Map.

<Editor type="exercise" lang="javascript">
<code>
let fruits = new Map();
console.log(fruits);
</code>

<solution>
let fruits = new Map();
fruits.set("fruitName", "apple");
console.log(fruits);
</solution>
