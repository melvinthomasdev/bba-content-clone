Let's use the **bracket notation**
to update the value of `103` key
to `"Mathematics"` in the `subjects` object.

<codeblock language="javascript" type="lesson">
<code>
const subjects = {
  101: "Physics",
  102: "Biology",
  103: "Chemistry"
};

subjects[103] = "Mathematics";

console.log(subjects);
</code>
</codeblock>
