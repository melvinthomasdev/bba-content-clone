Let's assume we have an object `person`
as shown in the editor.

Let's update the `age` property
to `23` using **dot notation**.

<codeblock language="javascript" type="lesson">
<code>
const person = {
  fullName: "Adam Smith",
  age: 37,
  isAdmin: true
};

person.age = 23;

console.log(person);
</code>
</codeblock>
