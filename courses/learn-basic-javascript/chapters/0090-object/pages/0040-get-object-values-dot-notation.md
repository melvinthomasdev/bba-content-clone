We can read the values of properties
of an object using the **dot notation**.

In the example given below,
we are reading the **value** stored
in the `name` **key**.

<codeblock language="javascript" type="lesson">
<code>
const person = {
  name: "Oliver",
  hobby: "Writing"
};

console.log(person.name);
</code>
</codeblock>
