Given is a h1 tag with
some text.

Using hsl color system change
it's `background-color`
to black `hsl(0, 0%, 0%)`
and font `color` to 
yellow `hsl(60, 80%, 90%)`.


<codeblock language="css" type="exercise" testMode="fixedInput">
<code>
<panel language="html">
<h1 id="heading">HSL color system</h1>
</panel>
<panel language="css">
// Write code here
</panel>
</code>
<solution>
#heading {
  background-color: hsl(0, 0%, 0%);
  color: hsl(60, 80%, 90%)
}
</solution>
</codeblock>
