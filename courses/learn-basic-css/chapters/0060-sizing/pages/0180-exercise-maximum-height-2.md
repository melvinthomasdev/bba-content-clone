Give a maximum height of `100vh`
to the `img` element:

<codeblock language="css" type="exercise" testMode="fixedInput">
<code>
<panel language="html">
<img src="mumbai-01.jpg" alt="Mumbai" />
</panel>
<panel language="css">

</panel>
</code>

  max-height: 100vh;
}
</solution>
</codeblock>
