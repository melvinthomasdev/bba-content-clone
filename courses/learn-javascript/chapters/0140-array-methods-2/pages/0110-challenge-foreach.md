Use the `forEach()` method
to display each element
of the array `names`.

For example:
```js
Input:["Sam", "Eve", "Linh", "Carlos"]

Output:
"Sam"
"Eve"
"Linh"
"Carlos"
```

<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const names = ["Adam", "Eve", "Oliver", "Sam"];
names.forEach();
</code>

<solution>
const names = ["Adam", "Eve", "Oliver", "Sam"];
names.forEach(name => console.log(name));
</solution>
</codeblock>
