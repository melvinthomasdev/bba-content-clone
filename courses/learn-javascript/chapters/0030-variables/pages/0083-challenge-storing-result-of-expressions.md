Store the product of
`length` and `width` in `area`.

For example
```js
Input: length = 5, width = 20

Output: 100
```

<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const length = 230;
const width = 100;

const area = ;

console.log(area);
</code>

<solution>
const length = 230;
const width = 100;

const area = length * width;

console.log(area);
</solution>
</codeblock>
