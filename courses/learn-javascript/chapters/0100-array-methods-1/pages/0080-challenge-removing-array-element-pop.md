Remove the last element of the array `iceCreams`.

<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const iceCreams = ["Cookies and Cream", "Vanilla", "Cookie Dough"];

console.log(iceCreams);
</code>

<solution>

iceCreams.pop();

console.log(iceCreams);
