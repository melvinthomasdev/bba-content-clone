Check and log to the console if the given value of 'userInput' is `NaN` or not.

For example:
```js
Input: userInput = "0a07"

Output: true
```
<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const userInput = "007";
console.log();
</code>
<solution>
const userInput = "007";
console.log(isNaN(userInput));
</solution>
</codeblock>
