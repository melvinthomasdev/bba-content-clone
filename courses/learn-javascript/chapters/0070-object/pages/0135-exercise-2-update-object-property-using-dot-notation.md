| Key            | Value   |
| -------------- | ------- |
| type           | Car     |
| numberOfWheels | 4       |
| fuel           | Petrol  |

Then, update the value for the
key `fuel` with `Diesel` using dot
notation.

Log the value of the object
`vehicle` to the console.
