Read the value of the `102` property
from the `subjects` object
using the **bracket notation**.

<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const subjects = {
  101: "Physics",
  102: "Biology",
  103: "Chemistry"
};

console.log();
</code>

<solution>
const subjects = {
  101: "Physics",
  102: "Biology",
  103: "Chemistry"
};

console.log(subjects[102]);
</solution>
</codeblock>
