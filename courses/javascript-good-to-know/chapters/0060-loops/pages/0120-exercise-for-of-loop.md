Use a `for...of` loop to print
all the **values** of the `fruits` array.

Use `console.log` to print the output.

<codeblock language="javascript" type="exercise" testMode="fixedInput">
<code>
const fruits = [
  "Apple",
  "Orange",
  "Banana"
]
</code>

<solution>
const fruits = [
  "Apple",
  "Orange",
  "Banana"
]

for (const fruit of fruits) {
  console.log(fruit);
}
</solution>
</codeblock>
