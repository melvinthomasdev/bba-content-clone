Remove the first element of the array `iceCreams`.

<Editor lang="javascript" type="exercise">
<code>
let iceCreams = ["Cookies and Cream", "Vanilla", "Cookie Dough"];

console.log(iceCreams);
</code>

<solution>

iceCreams.shift();

console.log(iceCreams);
