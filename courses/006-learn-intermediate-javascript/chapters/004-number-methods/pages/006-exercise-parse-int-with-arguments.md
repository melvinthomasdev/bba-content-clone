Use the `parseInt()` function
with `8` as the second argument
to convert the string
stored in the variable `octalNumber` to
an integer.

Use `console.log`
to show the result.

<Editor lang="javascript" type="exercise">
<code>
let octalNumber = "173";
console.log();
</code>

<solution>
let octalNumber = "173";
console.log(parseInt(octalNumber, 8));
</solution>
