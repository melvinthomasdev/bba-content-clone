In this exercise, add to the
hash, a key of **height** and
a value **6 ft**.

<codeblock language="ruby" type="exercise" testMode="fixedInput">
<code>
hash = {
  first_name: "John",
  last_name: "Smith",
  age: 25
}
# Write your code here
</code>

<solution>
hash = {
  first_name: "John",
  last_name: "Smith",
  age: 25
}

hash[:height] = "6 ft"

puts hash
</solution>
