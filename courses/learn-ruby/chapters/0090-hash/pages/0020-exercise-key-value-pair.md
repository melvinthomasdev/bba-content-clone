In this exercise,
create a hash using the
details given below:

|Key       |Value|
|--|--|
|first_name|John |
|last_name |Smith|
|age       |25   |

The keys of the hash
should be **first_name**,
**last_name** and **age**.

<codeblock language="ruby" type="exercise" testMode="fixedInput">
<code>
hash = {

}

puts hash
</code>

<solution>
hash = {
  "first_name" => "John",
  "last_name" => "Smith",
  "age" => 25
}

puts hash
</solution>
</codeblock>
