Ruby provides `.even?` method
to check if a number is even.

<codeblock language="ruby" type="lesson">
<code>
puts 2.even?

puts 21.even?
</code>
</codeblock>

Ruby also provides `.odd?` method
to check if a number is odd.

<codeblock language="ruby" type="lesson">
<code>
puts 2.odd?

puts 21.odd?
</code>
</codeblock>
