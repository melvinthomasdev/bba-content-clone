Ruby provides `start_with?`
to check if a string
starts with the given
string.

<codeblock language="ruby" type="lesson">
<code>
name = "John Smith"
puts name.start_with?("John")
</code>
</codeblock>

Here is another example:

<codeblock language="ruby" type="lesson">
<code>
name = "Chicago"
puts name.start_with?("Chi")
</code>
</codeblock>
