Ruby provides `include?`
to check if a string
includes another string.

<codeblock language="ruby" type="lesson">
<code>
name = "John Smith"
puts name.include?("Smith")
</code>
</codeblock>

Here is another example:

<codeblock language="ruby" type="lesson">
<code>
name = "Chicago"
puts name.include?("go")
</code>
</codeblock>
