Ruby can do simple math.

You can see the result using `puts`.

Run the code to see the result for addition:

<codeblock language="ruby" type="lesson">
<code>
puts 10 + 20
</code>
</codeblock>

Subtraction:

<codeblock language="ruby" type="lesson">
<code>
puts 30 - 10
</code>
</codeblock>

Multiplication:

<codeblock language="ruby" type="lesson">
<code>
puts 8 * 7
</code>
</codeblock>

Division:

<codeblock language="ruby" type="lesson">
<code>
puts 30/6
</code>
</codeblock>
