Display a new array on the screen
that contains all the elements of the
first array followed by all the
elements of the second array.

For example:
```ruby
Input:
array1 = [1, 2, 3]
array2 = [4, 5, 6]

Output:
[1, 2, 3, 4, 5, 6]
```
