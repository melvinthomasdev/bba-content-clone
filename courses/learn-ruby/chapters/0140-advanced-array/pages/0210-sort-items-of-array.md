We can use `sort` to
sort the elements in an array
in the ascending order.

<codeblock language="ruby" type="lesson">
<code>
array = [1,6,3,87,4,98]
sorted_array = array.sort
puts sorted_array
</code>
</codeblock>

Let's see one more example:

<codeblock language="ruby" type="lesson">
<code>
cities = %w(Chicago Seattle Boston Miami)
sorted_cities = cities.sort
puts sorted_cities
</code>
</codeblock>
