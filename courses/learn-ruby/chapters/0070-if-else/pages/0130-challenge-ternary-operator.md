If the length of **name** is
more than 4 letters,
then print **long name**.
Otherwise, print **small name**.
Use ternary operator to
solve this problem.

<codeblock language="ruby" type="exercise" testMode="fixedInput">
<code>
name = "Mary"
</code>

<solution>
name = "Mary"
puts name.length > 4 ? "long name" : "small name"
</solution>
</codeblock>
